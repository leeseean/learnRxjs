# learnRxjs

## 介绍

RxJS源码分析，记录了个人学习RxJS 源码的过程中的一些心得以及收获。

本项目希望对RxJS 做更进一步的探索与学习，RxJS 基础内容请参考RxJS 官网，[https://rxjs-dev.firebaseapp.com/](https://rxjs-dev.firebaseapp.com/)。
可能会有理解存在偏差的地方，欢迎提issue指出，共同学习，共同进步。
